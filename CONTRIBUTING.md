# Contribuindo

Obrigado pelo interesse em contribuir!

## Como contribuir

1. Faça um fork do repositório
2. Crie uma branch para sua feature (`git checkout -b feat/minha-feature`)
3. Faça suas alterações
4. Commit com mensagens semânticas em português (`git commit -m "feat: descrição"`)
5. Push para sua branch (`git push origin feat/minha-feature`)
6. Abra um Pull Request

## Convenções de commit

Usamos commits semânticos em português:

- `feat:` — nova funcionalidade
- `fix:` — correção de bug
- `docs:` — documentação
- `chore:` — manutenção e tarefas gerais
- `refactor:` — refatoração sem mudança de comportamento

## Reportando bugs

Abra uma [issue](https://github.com/LuisMIguelFurlanettoSousa/dotfiles/issues) com:

- Descrição do problema
- Passos para reproduzir
- Output de `lspci | grep VGA` e `cat /etc/os-release`

## Sugestões

Ideias de melhoria são bem-vindas! Abra uma issue com a tag `enhancement`.
